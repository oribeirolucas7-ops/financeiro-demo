FinanceFlow é uma aplicação web simples e moderna para organização financeira pessoal, desenvolvida utilizando HTML, CSS e JavaScript puro, sem dependências externas ou backend.
O objetivo do projeto é permitir o controle básico de entradas, saídas e saldo, com persistência dos dados no navegador.

Funcionalidades:
1.Cadastro de entradas e saídas financeiras
2.Cálculo automático de:
-Total de entradas
-Total de saídas
-Saldo final
-Exclusão de transações
3.Persistência de dados utilizando LocalStorage
4.Layout moderno em modo escuro
5.Interface responsiva (desktop e mobile)
6.Sem necessidade de login ou servidor
7.Tecnologias utilizadas:
-HTML5
-CSS3
-JavaScript (ES6+)
-LocalStorage (Web Storage API)
